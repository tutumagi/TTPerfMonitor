# TTPerfMonitor
A PerfMonitor for iOS from [RCTPerfMonitor](https://github.com/facebook/react-native/tree/master/React/Profiler)

### Usage:
```objc
#import "TTPerfMonitor.h"

[[TTPerfMonitor shareInstance] show];
```
